DWPF: Dark Web Pharma Framework (Investigative Tool)

Notes

This Dark Web Pharma Framework (DWPF) Investigative Tool was created using the open-source OSINT Framework by Justin Nordine (https://github.com/lockfale/osint-framework).
DWPIT leverages the OSINT framework by collecting information from several free sources and tools resources to aid pharmacutical investigators as they conduct investigations relating to the dark web and COVID-19 product being either advertised or sold on the Dark Web.
Ultimately, the goal is to help investigators navigate the Dark Web with these free Tool resources.
