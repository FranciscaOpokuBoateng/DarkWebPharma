OSINT Dark Web Investigative Framework (DWIF)

Notes

This Open-Source Intelligence (OSINT) Dark Web Investigative Framework (DWIF) was created using the open-source OSINT Framework by Justin Nordine (https://github.com/lockfale/osint-framework).
OSINT DWIF leverages the OSINT framework by collecting information from several free sources and tools resources to aid investigators in identifying counterfeit identification documents darknets and other information associates with the procurement of counterfeit identification on the Dark Web.
Ultimately, the goal is to help investigators navigate the Dark Web with these free OSINT resources.
